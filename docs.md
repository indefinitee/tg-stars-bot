«Вот бизнес-логика модуля "Звездочки":

Система работает периодами (раз в 2 месяца).

В каждом периоде сотрудник (Employee) имеет 3 голоса (звезды).

Голос — это запись в БД, связывающая отправителя, получателя и период. Вес голоса по умолчанию — 5.

Сотрудник не может голосовать за себя или дважды за одного человека в одном периоде.

HR-админ управляет списком сотрудников (активация/деактивация для голосования) и открывает/закрывает периоды.

Данные о сотрудниках синхронизируются из Bitrix24 (внешний источник истины).
